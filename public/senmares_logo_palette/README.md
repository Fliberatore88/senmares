- **Azul Marino Oscuro:** #1a3a52 (o similar) - Color principal, confianza y profesionalismo

## 💡 Guía de Uso

### Según el Fondo de tu Sitio Web:

1. **Fondo Blanco/Claro:**
   - Usa versiones con fondo transparente (elementos en azul marino y cyan)
   - O versión con fondo azul marino para crear contraste

2. **Fondo Oscuro/Azul Marino:**
   - Usa versiones con elementos blancos y cyan
   - O versión con fondo cyan para destacar

3. **Fondo de Color:**
   - Usa versiones con fondo transparente para máxima versatilidad
   - O versión con fondo sólido que contraste bien

### Según el Contexto:

- **Web:** Horizontal transparent para header, favicon transparent para browser tab
- **Mobile:** Favicon navy_bg como app icon, vertical para pantallas móviles
- **Impresos:** Hero transparent para folletos de alta calidad
- **Redes Sociales:** 
  - Facebook/LinkedIn Cover: banner_navy_bg.png
  - Instagram/Facebook Profile: favicon_transparent.png
  - Instagram Stories: vertical_navy_bg.png

## 📦 Formatos Disponibles

Todos los logos están en formato **PNG** con:
- Alta resolución
- Transparencia (donde corresponde)
- Optimizados para web e impresión
- Sin pérdida de calidad

---

## 🔧 Recomendaciones Técnicas

1. **Para Web:** 
   - Usa las versiones transparentes y optimiza con herramientas como TinyPNG
   - Considera crear versiones WebP para mejor rendimiento
   
2. **Para Impresión:**
   - Usa las versiones hero/grandes
   - Convierte a CMYK si es necesario
   - Asegúrate de tener al menos 300 DPI

3. **Para Apps:**
   - Genera los tamaños específicos de favicon que necesites (16x16, 32x32, etc.)
   - Usa la versión navy_bg para mejor visibilidad
